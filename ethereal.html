Het Project

Voor het vak programmeren 5 moest ik een website maken met het framework Laravel.

De website moest de mogelijkheid bevatten voor het aanmaken van een account. Daarnaast moest het mogelijk zijn voor iemand met een account om een post te maken. Op deze website was dat in versie van “plugins files” delen. Het maken van deze tekst was puur tekst omdat de plugin niks minder dan een concept was

Het Product

Het idee van Ethereal was het verkopen van plugins voor het maken van muziek. Deze plugins waren bijvoorbeeld voor reverb, compression, delay, etc.

Bij het maken van de opdracht is er gefocust op een van deze plugins genaamd Reflection. Reflection zou een compression plugin zijn.

Voor de plugin zijn in photoshop designs gemaakt, en vervolgens in een 3D model gemaakt in blender.
